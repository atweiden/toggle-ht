# toggle-ht

Toggle hyperthreading on x86 systems

## Synopsis

```sh
toggle-ht off
toggle-ht on
```

## Installation

```sh
install -Dm 755 toggle-ht -t /usr/bin
```

### Dependencies

- bash
- coreutils
